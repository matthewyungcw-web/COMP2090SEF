# Library Management System

Library Management System aims for managing books, borrowing activities, and records with a simple graphical user interface.

## Features

- Borrow Book
- Return Book
- Check Borrow Record
- Add Book (Admin Only)

## User Experience(normal user)

1. Run main.py
2. Login with correct credentials
3. Borrow book
4. Check borrow records
5. Return book
6. Logout

## User Experience(admin user)

1. Run main.py
2. Login with correct credentials
3. add book
4. view available books 
5. Logout
